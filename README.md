# Mc Lan UPNP
____________
 This mod allows your friends to join your world across the internet

Dedicated Server:
slap the mod in
give out your ip adress + ":" + port

Lan Server:
1. install the mod
2. open to lan
3. give them your public ip adress + ":" + port
4. enjoy
